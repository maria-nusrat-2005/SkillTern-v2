# UI Guidelines: Skilltern

This document specifies the UI development guidelines, responsive design requirements, and custom styling rules.

## 1. Tailwind Usage
- Utilize Tailwind utility classes instead of inline or styled components CSS.
- Adhere to the defined dark-mode themes (deep slates, dark gray backgrounds, emerald success badges).

## 2. Layout & Responsiveness
- Enforce mobile-first layouts using Tailwind responsive prefixes (`sm:`, `md:`, `lg:`).
- Keep component components reusable and clean.
- Ensure all forms use React Hook Form to prevent general state re-renders.
